# Sistema de Rastreamento de Pedidos

Aplicação web desenvolvida em Python utilizando Flask para consulta de pedidos e rastreamento de entregas.

## Funcionalidades

* Consulta por número do pedido
* Consulta manual por CNPJ e Nota Fiscal
* Integração com Google Sheets
* Integração com plataforma SSW
* Interface web responsiva
* Consulta automática de status de entrega

## Tecnologias Utilizadas

* Python
* Flask
* Pandas
* Requests
* BeautifulSoup
* HTML
* CSS

## Fluxo do Sistema

1. Cliente informa o número do pedido
2. Sistema consulta a planilha Google Sheets
3. Obtém automaticamente CNPJ e Nota Fiscal
4. Consulta o rastreamento na SSW
5. Exibe o status atualizado da entrega

## Objetivo

Projeto desenvolvido para automatizar o acompanhamento logístico de pedidos, reduzindo consultas manuais e melhorando a experiência do cliente.
